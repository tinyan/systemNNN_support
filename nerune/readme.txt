効果音変換ツール nerune.exe ver 1.00


システム効果音
wav\sys\*.wav -> vaw\sys\*.vaw

スクリプト効果音
wav\se\*.wav -> vaw\se\*.vaw

音楽
bgm\*.ogg -> wgq\*.wgq
bgm\*.mml -> wgq\*.mml

音声 (nnndir\charaname.txtを参照して変換を行う)
cdwave\*.wav -> cdvaw\[name]\*.vaw
cdwave\*.ogg -> cdvaw\[name]\*.vaw   (cdwave\*.wavも必要)


固める機能 (nnndir\charaname.txtを参照して変換を行う)
cdvaw\[name]\*.vaw -> cdvaw\[name].vtb + cdvaw\[name].vpk
